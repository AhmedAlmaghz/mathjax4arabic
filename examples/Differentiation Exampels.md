## 📚 الشرح

### تعريف الاشتقاق

الاشتقاق (Differentiation) هو أحد المفاهيم الأساسية في التفاضل، ويُعبّر عن معدل تغيّر دالة بالنسبة لمتغيرها. تُعرَّف مشتقة الدالة $f$ عند نقطة $x$ باستخدام النهاية التالية:

**الصيغة:**
$f'(x)=\lim_{h \to 0} \frac{f(x+h)-f(x)}{h}$

## ✏️ الأمثلة المحلولة (58 مثالًا)

### مثال 1

**القانون/النظرية:** تعريف المشتقة

**المطلوب:**

$f(x) = x^2 \quad \text{أوجد } f'(x)$

**الحل:**

1. $f'(x)=\lim_{h\to 0}\dfrac{f(x+h)-f(x)}{h}$
2. $f(x+h)=(x+h)^2 = x^2+2xh+h^2$
3. $f(x+h)-f(x) = 2xh+h^2 = h(2x+h)$
4. $\dfrac{f(x+h)-f(x)}{h} = 2x+h$
5. $f'(x)=\lim_{h\to 0}(2x+h)=2x$

**الناتج:** $f'(x) = 2x$

---

### مثال 2

**القانون/النظرية:** تعريف المشتقة مع الترشيد

**المطلوب:**

$f(x) = \sqrt{x} \quad \text{أوجد } f'(x)$

**الحل:**

1. $f'(x)=\lim_{h\to 0}\dfrac{\sqrt{x+h}-\sqrt{x}}{h}$
2. نضرب في المرافق: $\dfrac{(\sqrt{x+h}-\sqrt{x})(\sqrt{x+h}+\sqrt{x})}{h(\sqrt{x+h}+\sqrt{x})}$
3. $= \dfrac{(x+h)-x}{h(\sqrt{x+h}+\sqrt{x})} = \dfrac{1}{\sqrt{x+h}+\sqrt{x}}$
4. $f'(x)=\dfrac{1}{2\sqrt{x}}$

**الناتج:** $f'(x) = \frac{1}{2\sqrt{x}}$

---

### مثال 3

**القانون/النظرية:** تعريف المشتقة

**المطلوب:**

$f(x) = \frac{1}{x} \quad \text{أوجد } f'(x)$

**الحل:**

1. $f'(x)=\lim_{h\to 0}\dfrac{\frac{1}{x+h}-\frac{1}{x}}{h}$
2. $= \lim_{h\to 0}\dfrac{x-(x+h)}{hx(x+h)} = \lim_{h\to 0}\dfrac{-h}{hx(x+h)}$
3. $= \lim_{h\to 0}\dfrac{-1}{x(x+h)} = -\dfrac{1}{x^2}$

**الناتج:** $f'(x) = -\frac{1}{x^2}$

---

### مثال 4

**القانون/النظرية:** تعريف المشتقة

**المطلوب:**

$f(x) = x^3 \quad \text{أوجد } f'(x)$

**الحل:**

1. $(x+h)^3 = x^3 + 3x^2h + 3xh^2 + h^3$
2. $f(x+h) - f(x) = 3x^2h + 3xh^2 + h^3 = h(3x^2 + 3xh + h^2)$
3. $\dfrac{f(x+h)-f(x)}{h} = 3x^2 + 3xh + h^2$
4. $f'(x) = \lim_{h\to 0}(3x^2 + 3xh + h^2) = 3x^2$

**الناتج:** $f'(x) = 3x^2$

---

### مثال 5

**القانون/النظرية:** تعريف المشتقة مع متطابقات

**المطلوب:**

$f(x) = \sin x \quad \text{أوجد } f'(x)$

**الحل:**

1. $f'(x) = \lim_{h\to 0}\dfrac{\sin(x+h) - \sin x}{h}$
2. $\sin(x+h) - \sin x = 2\cos\left(x+\dfrac{h}{2}\right)\sin\left(\dfrac{h}{2}\right)$
3. $= \lim_{h\to 0} \cos\left(x+\dfrac{h}{2}\right) \cdot \dfrac{\sin(h/2)}{h/2}$
4. بما أن $\lim_{t\to 0}\dfrac{\sin t}{t} = 1$:
5. $f'(x) = \cos x \cdot 1 = \cos x$

**الناتج:** $f'(x) = \cos x$

---

**الناتج:** $f'(x) = x^x(\ln x + 1)$

### مثال 43

**القانون/النظرية:** الجمع

**المطلوب:**
