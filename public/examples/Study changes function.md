### دراسة الدوال

دراسة تزايد وتناقص وقيم قصوى وتقعر الدوال

## ✏️ الأمثلة المحلولة (16 مثالًا)

### مثال 1

**القانون/النظرية:** 

**المطلوب:**

$f(x) = x^3 - 3x + 2$

**الحل:**

1. نجد المشتقة الأولى: $f'(x) = 3x^2 - 3 = 3(x^2 - 1) = 3(x-1)(x+1)$
2. نجد النقاط الحرجة: $f'(x) = 0$
3. $3(x-1)(x+1) = 0 \Rightarrow x = -1$ أو $x = 1$
4. ندرس إشارة $f'(x)$ في كل فترة:
5. • في $(-\infty, -1)$: نختار $x = -2$، $f'(-2) = 3(4-1) = 9 > 0$ ↗
6. • في $(-1, 1)$: نختار $x = 0$، $f'(0) = -3 < 0$ ↘
7. • في $(1, +\infty)$: نختار $x = 2$، $f'(2) = 3(4-1) = 9 > 0$ ↗

**الناتج:** $f \text{ متزايدة على } (-\infty, -1) \cup (1, +\infty) \text{ ومتناقصة على } (-1, 1)$

### مثال 2

**القانون/النظرية:** 

**المطلوب:**

$f(x) = \frac{x}{x^2 + 1}$

**الحل:**

1. نجد المشتقة باستخدام قاعدة القسمة:
2. $f'(x) = \dfrac{1 \cdot (x^2+1) - x \cdot 2x}{(x^2+1)^2} = \dfrac{x^2 + 1 - 2x^2}{(x^2+1)^2}$
3. $f'(x) = \dfrac{1 - x^2}{(x^2+1)^2} = \dfrac{(1-x)(1+x)}{(x^2+1)^2}$
4. المقام $(x^2+1)^2 > 0$ دائمًا
5. إشارة $f'(x)$ تعتمد على البسط $(1-x)(1+x)$:
6. • $f'(x) > 0$ عندما $-1 < x < 1$
7. • $f'(x) < 0$ عندما $x < -1$ أو $x > 1$

**الناتج:** $f \text{ متزايدة على } (-1, 1) \text{ ومتناقصة على } (-\infty, -1) \cup (1, +\infty)$

### مثال 3

**القانون/النظرية:** 

**المطلوب:**

$f(x) = x^3 + x$

**الحل:**

1. $f'(x) = 3x^2 + 1$
2. بما أن $x^2 \geq 0$ لكل $x$:
3. $f'(x) = 3x^2 + 1 \geq 1 > 0$ لكل $x \in \mathbb{R}$
4. المشتقة موجبة دائمًا

**الناتج:** $f \text{ متزايدة تمامًا على } \mathbb{R}$

### مثال 4

**القانون/النظرية:** 

**المطلوب:**

$f(x) = xe^{-x}$

**الحل:**

1. نستخدم قاعدة الضرب:
2. $f'(x) = 1 \cdot e^{-x} + x \cdot (-e^{-x}) = e^{-x}(1 - x)$
3. بما أن $e^{-x} > 0$ دائمًا:
4. إشارة $f'(x)$ = إشارة $(1 - x)$
5. • $f'(x) > 0$ عندما $x < 1$
6. • $f'(x) < 0$ عندما $x > 1$

**الناتج:** $f \text{ متزايدة على } (-\infty, 1) \text{ ومتناقصة على } (1, +\infty)$

### مثال 5

**القانون/النظرية:** extrema

**المطلوب:**

$f(x) = x^3 - 6x^2 + 9x + 1$

**الحل:**

1. $f'(x) = 3x^2 - 12x + 9 = 3(x^2 - 4x + 3) = 3(x-1)(x-3)$
2. النقاط الحرجة: $x = 1$ و $x = 3$
3. نستخدم اختبار المشتقة الثانية:
4. $f''(x) = 6x - 12$
5. • عند $x = 1$: $f''(1) = 6 - 12 = -6 < 0$ ← قيمة عظمى محلية
6. • عند $x = 3$: $f''(3) = 18 - 12 = 6 > 0$ ← قيمة صغرى محلية
7. $f(1) = 1 - 6 + 9 + 1 = 5$
8. $f(3) = 27 - 54 + 27 + 1 = 1$

**الناتج:** $\text{قيمة عظمى محلية } f(1) = 5 \text{، قيمة صغرى محلية } f(3) = 1$

### مثال 6

**القانون/النظرية:** extrema

**المطلوب:**

$f(x) = x^3 - 3x + 1 \quad \text{على } [-2, 2]$

**الحل:**

1. $f'(x) = 3x^2 - 3 = 3(x^2 - 1) = 3(x-1)(x+1)$
2. النقاط الحرجة في $[-2, 2]$: $x = -1$ و $x = 1$
3. نحسب قيم الدالة عند النقاط الحرجة وطرفي الفترة:
4. • $f(-2) = -8 + 6 + 1 = -1$
5. • $f(-1) = -1 + 3 + 1 = 3$
6. • $f(1) = 1 - 3 + 1 = -1$
7. • $f(2) = 8 - 6 + 1 = 3$
8. نقارن القيم: أكبر قيمة = $3$، أصغر قيمة = $-1$

**الناتج:** $\text{القيمة العظمى المطلقة } = 3 \text{ عند } x = -1, 2 \text{؛ الصغرى } = -1 \text{ عند } x = -2, 1$

### مثال 7

**القانون/النظرية:** extrema

**المطلوب:**

$f(x) = x^3$

**الحل:**

1. $f'(x) = 3x^2$
2. النقطة الحرجة: $f'(x) = 0 \Rightarrow x = 0$
3. ندرس إشارة $f'(x)$:
4. $f'(x) = 3x^2 \geq 0$ لكل $x$
5. المشتقة لا تغير إشارتها عند $x = 0$
6. (موجبة قبل وبعد $x = 0$)

**الناتج:** $\text{لا توجد قيم قصوى محلية؛ النقطة } (0, 0) \text{ نقطة انعطاف}$

### مثال 8

**القانون/النظرية:** extrema

**المطلوب:**

$\text{مستطيل محيطه } 20 \text{ م. أوجد أبعاده ليكون أكبر مساحة}$

**الحل:**

1. نفرض الطول $x$ والعرض $y$
2. المحيط: $2x + 2y = 20 \Rightarrow y = 10 - x$
3. المساحة: $A(x) = xy = x(10-x) = 10x - x^2$
4. حيث $0 < x < 10$
5. $A'(x) = 10 - 2x$
6. $A'(x) = 0 \Rightarrow x = 5$
7. $A''(x) = -2 < 0$ ← قيمة عظمى
8. إذن $x = 5$ و $y = 10 - 5 = 5$

**الناتج:** $\text{مربع ضلعه } 5 \text{ م يعطي أكبر مساحة } = 25 \text{ م}^2$

### مثال 9

**القانون/النظرية:** extrema

**المطلوب:**

$\text{صندوق بدون غطاء، قاعدته مربعة، حجمه } 32 \text{ م}^3\text{. أوجد أبعاده لتقليل مساحة السطح}$

**الحل:**

1. نفرض ضلع القاعدة $x$ والارتفاع $h$
2. الحجم: $x^2 h = 32 \Rightarrow h = \dfrac{32}{x^2}$
3. مساحة السطح (بدون غطاء):
4. $S = x^2 + 4xh = x^2 + 4x \cdot \dfrac{32}{x^2} = x^2 + \dfrac{128}{x}$
5. $S'(x) = 2x - \dfrac{128}{x^2} = \dfrac{2x^3 - 128}{x^2}$
6. $S'(x) = 0 \Rightarrow 2x^3 = 128 \Rightarrow x^3 = 64 \Rightarrow x = 4$
7. $h = \dfrac{32}{16} = 2$
8. $S''(x) = 2 + \dfrac{256}{x^3}$، وعند $x=4$: $S''(4) > 0$ ← قيمة صغرى

**الناتج:** $\text{القاعدة } 4 \times 4 \text{ م والارتفاع } 2 \text{ م}$

### مثال 10

**القانون/النظرية:** concavity

**المطلوب:**

$f(x) = x^3 - 3x^2 + 2$

**الحل:**

1. $f'(x) = 3x^2 - 6x$
2. $f''(x) = 6x - 6 = 6(x - 1)$
3. نقطة الانعطاف المحتملة: $f''(x) = 0 \Rightarrow x = 1$
4. ندرس إشارة $f''(x)$:
5. • في $(-\infty, 1)$: $f''(x) < 0$ ← مقعر للأسفل ∩
6. • في $(1, +\infty)$: $f''(x) > 0$ ← مقعر للأعلى ∪
7. $f''(x)$ تغير إشارتها عند $x = 1$
8. $f(1) = 1 - 3 + 2 = 0$

**الناتج:** $\text{نقطة انعطاف عند } (1, 0) \text{؛ مقعر للأسفل على } (-\infty, 1) \text{ ومقعر للأعلى على } (1, +\infty)$

### مثال 11

**القانون/النظرية:** concavity

**المطلوب:**

$f(x) = x^4 - 6x^2 + 8x + 1$

**الحل:**

1. $f'(x) = 4x^3 - 12x + 8$
2. $f''(x) = 12x^2 - 12 = 12(x^2 - 1) = 12(x-1)(x+1)$
3. نقاط الانعطاف المحتملة: $x = -1$ و $x = 1$
4. ندرس إشارة $f''(x)$:
5. • في $(-\infty, -1)$: $f''(-2) = 12(3) > 0$ ← ∪
6. • في $(-1, 1)$: $f''(0) = -12 < 0$ ← ∩
7. • في $(1, +\infty)$: $f''(2) = 12(3) > 0$ ← ∪

**الناتج:** $\text{نقطتا انعطاف عند } x = -1 \text{ و } x = 1$

### مثال 12

**القانون/النظرية:** concavity

**المطلوب:**

$f(x) = x \ln x \quad (x > 0)$

**الحل:**

1. $f'(x) = \ln x + x \cdot \dfrac{1}{x} = \ln x + 1$
2. $f''(x) = \dfrac{1}{x}$
3. بما أن $x > 0$: $f''(x) = \dfrac{1}{x} > 0$ دائمًا
4. لا توجد نقاط انعطاف

**الناتج:** $f \text{ مقعرة للأعلى على } (0, +\infty) \text{ بدون نقاط انعطاف}$

### مثال 13

**القانون/النظرية:** full-study

**المطلوب:**

$f(x) = x^3 - 3x^2 - 9x + 5$

**الحل:**

1. **المجال:** $D_f = \mathbb{R}$
2. **التقاطعات:** $f(0) = 5$، $f(x) = 0$ (معقدة)
3. **المشتقة الأولى:** $f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)$
4. **النقاط الحرجة:** $x = -1$ و $x = 3$
5. **التزايد والتناقص:**
6. • $f$ متزايدة على $(-\infty, -1) \cup (3, +\infty)$
7. • $f$ متناقصة على $(-1, 3)$
8. **القيم القصوى:** $f(-1) = -1-3+9+5 = 10$ (عظمى)، $f(3) = 27-27-27+5 = -22$ (صغرى)
9. **المشتقة الثانية:** $f''(x) = 6x - 6 = 6(x-1)$
10. **التقعر:** مقعر للأسفل على $(-\infty, 1)$، للأعلى على $(1, +\infty)$
11. **نقطة الانعطاف:** $(1, f(1)) = (1, -6)$
12. **السلوك عند اللانهاية:** $\lim_{x \to \pm\infty} f(x) = \pm\infty$

**الناتج:** $\text{انظر جدول التغيرات والرسم}$

### مثال 14

**القانون/النظرية:** full-study

**المطلوب:**

$f(x) = \frac{x^2}{x-1}$

**الحل:**

1. **المجال:** $D_f = \mathbb{R} \setminus \{1\}$
2. **التقاطعات:** $f(0) = 0$
3. **الخطوط المقاربة:**
4. • رأسي: $x = 1$
5. • مائل: $f(x) = x + 1 + \dfrac{1}{x-1}$، إذن $y = x + 1$
6. **المشتقة:** $f'(x) = \dfrac{2x(x-1) - x^2}{(x-1)^2} = \dfrac{x^2 - 2x}{(x-1)^2} = \dfrac{x(x-2)}{(x-1)^2}$
7. **النقاط الحرجة:** $x = 0$ و $x = 2$
8. **القيم:** $f(0) = 0$ (عظمى محلية)، $f(2) = 4$ (صغرى محلية)
9. **التزايد:** على $(-\infty, 0) \cup (2, +\infty)$
10. **التناقص:** على $(0, 1) \cup (1, 2)$

**الناتج:** $\text{انظر جدول التغيرات}$

### مثال 15

**القانون/النظرية:** full-study

**المطلوب:**

$f(x) = x^2 e^{-x}$

**الحل:**

1. **المجال:** $D_f = \mathbb{R}$
2. **المشتقة:** $f'(x) = 2xe^{-x} + x^2(-e^{-x}) = e^{-x}(2x - x^2) = xe^{-x}(2-x)$
3. **النقاط الحرجة:** $x = 0$ و $x = 2$
4. **دراسة الإشارة:** $e^{-x} > 0$ دائمًا
5. • $f'(x) > 0$ عندما $0 < x < 2$
6. • $f'(x) < 0$ عندما $x < 0$ أو $x > 2$
7. **القيم القصوى:** $f(0) = 0$ (صغرى)، $f(2) = 4e^{-2}$ (عظمى)
8. **السلوك:** $\lim_{x \to -\infty} f(x) = +\infty$، $\lim_{x \to +\infty} f(x) = 0$

**الناتج:** $\text{صغرى عند } x=0 \text{، عظمى عند } x=2 \text{، خط مقارب أفقي } y=0$

### مثال 16

**القانون/النظرية:** full-study

**المطلوب:**

$f(x) = \sin x + \cos x \quad \text{على } [0, 2\pi]$

**الحل:**

1. نكتب: $f(x) = \sqrt{2}\sin\left(x + \dfrac{\pi}{4}\right)$
2. **المشتقة:** $f'(x) = \cos x - \sin x = \sqrt{2}\cos\left(x + \dfrac{\pi}{4}\right)$
3. **النقاط الحرجة:** $f'(x) = 0 \Rightarrow \cos x = \sin x$
4. $x = \dfrac{\pi}{4}$ أو $x = \dfrac{5\pi}{4}$
5. **القيم:**
6. • $f\left(\dfrac{\pi}{4}\right) = \sqrt{2}$ (عظمى)
7. • $f\left(\dfrac{5\pi}{4}\right) = -\sqrt{2}$ (صغرى)
8. **القيم عند الأطراف:** $f(0) = 1$، $f(2\pi) = 1$

**الناتج:** $\text{العظمى المطلقة } \sqrt{2} \text{ عند } x = \frac{\pi}{4} \text{، الصغرى } -\sqrt{2} \text{ عند } x = \frac{5\pi}{4}$
