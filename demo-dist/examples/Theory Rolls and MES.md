## 📚 الشرح

### نظرية رول

إذا كانت f متصلة على [a,b] وقابلة للاشتقاق على (a,b) وكان f(a) = f(b)

## ✏️ الأمثلة المحلولة (10 مثالًا)

### مثال 1

**القانون/النظرية:** تحقق من شروط نظرية رول

**المطلوب:**

$f(x) = x^2 - 4x + 3 \quad \text{على الفترة } [1, 3]$

**الحل:**

1. نتحقق من الشروط:
2. ① الاتصال: $f(x)$ كثيرة حدود، إذن متصلة على $[1,3]$ ✓
3. ② الاشتقاقية: $f(x)$ كثيرة حدود، إذن قابلة للاشتقاق على $(1,3)$ ✓
4. ③ تساوي الطرفين: $f(1) = 1-4+3 = 0$ و $f(3) = 9-12+3 = 0$ ✓
5. الشروط متحققة، نوجد $c$:
6. $f'(x) = 2x - 4$
7. نحل $f'(c) = 0$: $2c - 4 = 0 \Rightarrow c = 2$
8. نتحقق: $c = 2 \in (1, 3)$ ✓

**الناتج:** $c = 2$

### مثال 2

**القانون/النظرية:** إيجاد نقطة رول

**المطلوب:**

$f(x) = \sin x \quad \text{على الفترة } [0, \pi]$

**الحل:**

1. نتحقق من الشروط:
2. ① $\sin x$ متصلة على $[0, \pi]$ ✓
3. ② $\sin x$ قابلة للاشتقاق على $(0, \pi)$ ✓
4. ③ $f(0) = \sin 0 = 0$ و $f(\pi) = \sin \pi = 0$ ✓
5. نوجد $c$: $f'(x) = \cos x$
6. نحل $\cos c = 0$ في الفترة $(0, \pi)$
7. $c = \dfrac{\pi}{2}$

**الناتج:** $c = \frac{\pi}{2}$

---

### مثال 3

**القانون/النظرية:** نظرية رول مع دالة كسرية

**المطلوب:**

$f(x) = x^3 - 3x \quad \text{على الفترة } [-\sqrt{3}, \sqrt{3}]$

**الحل:**

1. نتحقق من الشروط:
2. ① $f(x)$ كثيرة حدود، متصلة ✓
3. ② قابلة للاشتقاق ✓
4. ③ $f(-\sqrt{3}) = -3\sqrt{3} + 3\sqrt{3} = 0$
5.    $f(\sqrt{3}) = 3\sqrt{3} - 3\sqrt{3} = 0$ ✓
6. نوجد $c$: $f'(x) = 3x^2 - 3$
7. $3c^2 - 3 = 0 \Rightarrow c^2 = 1 \Rightarrow c = \pm 1$
8. كلاهما في الفترة $(-\sqrt{3}, \sqrt{3})$

**الناتج:** $c = -1 \quad \text{أو} \quad c = 1$

---

### مثال 4

**القانون/النظرية:** عدم تحقق نظرية رول

**المطلوب:**

$f(x) = |x| \quad \text{على الفترة } [-1, 1]$

**الحل:**

1. نتحقق من الشروط:
2. ① $|x|$ متصلة على $[-1, 1]$ ✓
3. ② عند $x = 0$: النهاية من اليمين $= 1$، من اليسار $= -1$
4.    إذن $f(x)$ غير قابلة للاشتقاق عند $x = 0$ ✗
5. ③ $f(-1) = 1$ و $f(1) = 1$ ✓
6. الشرط الثاني غير متحقق

**الناتج:** $\text{نظرية رول لا تنطبق لأن الدالة غير قابلة للاشتقاق عند } x = 0$

### مثال 5

**القانون/النظرية:** تطبيق نظرية رول لإثبات وجود جذر

**المطلوب:**

$\text{أثبت أن المعادلة } x^3 + x - 1 = 0 \text{ لها جذر واحد فقط}$

**الحل:**

1. نفرض $f(x) = x^3 + x - 1$
2. نلاحظ: $f(0) = -1 < 0$ و $f(1) = 1 > 0$
3. بنظرية القيمة الوسيطة، يوجد جذر في $(0, 1)$
4. لإثبات الوحدانية: نفرض وجود جذرين $a < b$
5. بنظرية رول، يوجد $c \in (a, b)$ حيث $f'(c) = 0$
6. لكن $f'(x) = 3x^2 + 1 > 0$ لكل $x$ (لا يساوي صفرًا أبدًا)
7. تناقض! إذن الجذر وحيد.

**الناتج:** $\text{المعادلة لها جذر واحد فقط في } (0, 1)$

### مثال 6

**القانون/النظرية:** تطبيق مباشر لنظرية القيمة المتوسطة

**المطلوب:**

$f(x) = x^2 \quad \text{على الفترة } [1, 3]$

**الحل:**

1. نتحقق من الشروط: $f(x) = x^2$ متصلة وقابلة للاشتقاق ✓
2. نحسب ميل الوتر:
3. $\dfrac{f(3) - f(1)}{3 - 1} = \dfrac{9 - 1}{2} = 4$
4. نوجد $c$: $f'(x) = 2x$
5. $f'(c) = 4 \Rightarrow 2c = 4 \Rightarrow c = 2$
6. نتحقق: $c = 2 \in (1, 3)$ ✓

**الناتج:** $c = 2$

### مثال 7

**القانون/النظرية:** القيمة المتوسطة مع دالة جذرية

**المطلوب:**

$f(x) = \sqrt{x} \quad \text{على الفترة } [0, 4]$

**الحل:**

1. $f(x) = \sqrt{x}$ متصلة على $[0, 4]$ وقابلة للاشتقاق على $(0, 4)$
2. ميل الوتر: $\dfrac{f(4) - f(0)}{4 - 0} = \dfrac{2 - 0}{4} = \dfrac{1}{2}$
3. $f'(x) = \dfrac{1}{2\sqrt{x}}$
4. نحل: $\dfrac{1}{2\sqrt{c}} = \dfrac{1}{2}$
5. $2\sqrt{c} = 2 \Rightarrow \sqrt{c} = 1 \Rightarrow c = 1$

**الناتج:** $c = 1$

### مثال 8

**القانون/النظرية:** القيمة المتوسطة مع دالة مثلثية

**المطلوب:**

$f(x) = \cos x \quad \text{على الفترة } [0, \pi]$

**الحل:**

1. $\cos x$ متصلة وقابلة للاشتقاق ✓
2. ميل الوتر: $\dfrac{\cos \pi - \cos 0}{\pi - 0} = \dfrac{-1 - 1}{\pi} = -\dfrac{2}{\pi}$
3. $f'(x) = -\sin x$
4. نحل: $-\sin c = -\dfrac{2}{\pi}$
5. $\sin c = \dfrac{2}{\pi} \approx 0.637$
6. $c = \arcsin\left(\dfrac{2}{\pi}\right) \approx 0.69$

**الناتج:** $c = \arcsin\left(\frac{2}{\pi}\right) \approx 0.69$

### مثال 9

**القانون/النظرية:** تطبيق القيمة المتوسطة في السرعة

**المطلوب:**

$\text{سيارة قطعت } 120 \text{ كم في ساعتين. أثبت أن سرعتها كانت } 60 \text{ كم/س في لحظة ما}$

**الحل:**

1. نفرض $s(t)$ موضع السيارة بالكيلومتر عند الزمن $t$ بالساعات
2. $s(0) = 0$ و $s(2) = 120$
3. بنظرية القيمة المتوسطة، يوجد $c \in (0, 2)$ حيث:
4. $s'(c) = \dfrac{s(2) - s(0)}{2 - 0} = \dfrac{120 - 0}{2} = 60$
5. أي أن السرعة اللحظية عند اللحظة $c$ تساوي $60$ كم/س

**الناتج:** $\text{نعم، توجد لحظة } t = c \text{ كانت فيها السرعة } 60 \text{ كم/س}$

### مثال 10

**القانون/النظرية:** إثبات متباينة باستخدام القيمة المتوسطة

**المطلوب:**

$\text{أثبت أن } |\sin a - \sin b| \leq |a - b| \text{ لكل } a, b$

**الحل:**

1. نفرض $f(x) = \sin x$ على الفترة $[a, b]$ (أو $[b, a]$)
2. بنظرية القيمة المتوسطة، يوجد $c$ بين $a$ و $b$ حيث:
3. $\dfrac{\sin b - \sin a}{b - a} = \cos c$
4. إذن: $|\sin b - \sin a| = |\cos c| \cdot |b - a|$
5. بما أن $|\cos c| \leq 1$ لكل $c$:
6. $|\sin b - \sin a| \leq |b - a|$

**الناتج:** $|\sin a - \sin b| \leq |a - b| \quad \text{لكل } a, b \in \mathbb{R}$
